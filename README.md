# 💰 Finance Dashboard UI

A modern and interactive **Finance Dashboard** built using React and Tailwind CSS.
This project demonstrates clean UI design, state management, and frontend architecture for tracking financial activities.

---

## 🚀 Live Features

### 📊 Dashboard Overview

* Total Balance, Income, Expenses cards
* Time-based chart (Line chart)
* Category-based chart (Pie chart)

### 📋 Transactions Management

* View all transactions
* Search / filter transactions
* Add / Delete transactions (Admin only)

### 🔐 Role-Based UI

* Viewer → Can only view data
* Admin → Can add/delete transactions

### 📈 Insights Section

* Total expenses calculation
* Transaction count
* Basic financial observations

### 💾 Data Persistence

* Data stored using LocalStorage

### 📱 Responsive Design

* Works on desktop, tablet, and mobile

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 📊 Chart.js
* 🎬 Framer Motion (for animations)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/finance-dashboard.git
cd finance-dashboard
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the project

```bash
npm run dev
```

## 🧠 Approach

* Used **Context API** for centralized state management
* Designed reusable components for scalability
* Implemented **role-based UI logic** without backend
* Used **mock data + localStorage** for persistence
* Focused on clean UI, readability, and user experience
